## Description

최근 수요가 늘어난 인생네컷을 사람들끼리 공유할 수 있는 웹 어플리케이션이다.

전체적인 서비스는 인스타그램에서 모티브하였다.

웹 풀스택 개발을 경험해보기 위해 진행하게 되었다.

### fuction

- 회원가입, 로그인
- 프로필 설정
- 이미지 업로드
- 댓글, 좋아요
- 팔로우
